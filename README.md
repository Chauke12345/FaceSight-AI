# 🤖 FaceSight AI

## Intelligent Facial Analytics Platform

FaceSight AI is a browser-based computer vision application built with Django, JavaScript, TensorFlow.js, and face-api.js.

The application performs real-time facial analysis directly in the user's browser, including face detection, age estimation, gender prediction, and emotion recognition.

The project demonstrates the integration of artificial intelligence, machine learning models, and modern web technologies to create an interactive AI-powered application.

---

## 🚀 Features

### 📷 Real-Time Camera Analysis
- Access webcam directly from the browser
- Detect faces in real time
- Display AI analysis results instantly

### 🖼️ Image Upload Analysis
- Upload an image for facial analysis
- Preview uploaded images
- Run AI detection on uploaded photos

### 😀 Face Detection
- Detect one or multiple faces using computer vision models

### 🎂 Age Estimation
- Predict approximate age based on facial features

### 🚻 Gender Prediction
- Estimate gender using AI facial analysis

### 😊 Emotion Recognition
Detect facial expressions including:

- Happy
- Sad
- Angry
- Surprised
- Fearful
- Disgusted
- Neutral

### 🔄 Reset Functionality
- Clear uploaded images
- Reset AI results
- Return application to default state
